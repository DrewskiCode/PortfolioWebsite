Ranch Management · UX + iOS Prototype

Stable

A ranch management app that replaces scattered, verbal task updates with a clear, centralized system for daily operations.

Problem

Ranches move fast and rely on word-of-mouth status updates. Tasks get repeated, critical info gets buried, and time is wasted chasing answers.

Solution

Stable organizes tasks, livestock notes, and ranch updates in one place, making progress visible and easy to share across a team.

Key Use Cases

Daily Task Tracking
Horse Medical History
Team Status Updates
Ranch Operations

Process

  • Empathized with real ranch workflows and bottlenecks.
  • Defined the core problem around task visibility.
  • Designed wireframes to speed up development.
  • Built iOS prototypes for key flows.

Evaluation

Presented to ranch workers for feedback on usability, workflow clarity, and needed features. Focused on direct manipulation and consistency.

Live Web Test

To prototype features and gather real user feedback, I’m currently running a web app with similar functionality. When live, it’s available at poniesponies.com .
